# Sarcasm & Emotion Detection using NLP

This project detects sarcasm and emotions from text using Transformer-based models.
The pipeline first checks whether a sentence is sarcastic; if not, it predicts the dominant emotion.

## Features
- Sarcasm classification
- Emotion detection
- Streamlit-based UI
- Real-time predictions

## Tech Stack
- Python
- PyTorch
- Hugging Face Transformers
- Streamlit

## How to Run
pip install -r requirements.txt
streamlit run app.py
